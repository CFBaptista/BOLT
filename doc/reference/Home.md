# Home

## Project

BOLT (Boltzmann Operator Lattice Toolkit) is a C++ library for Computational Fluid Dynamics based on the Lattice Boltzmann Method.

## Documentation

This part of the documentation forms the public API reference of BOLT, developed using the C++23 standard.
